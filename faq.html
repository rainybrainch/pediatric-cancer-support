ヘルプ・FAQ

よくある質問・トラブルシューティング
📖 基本
ヴィタリア転生録ってなに？
小児がんを経験した方の健康習慣を、RPGの世界観で楽しく続けるためのアプリです。運動・食事・QOLを記録すると、キャラ「カイト」「セラ」「ルナ」「ピクセ」が成長し、ボスに挑戦できます。
医療行為ではなく、研究目的の自己モニタリングと行動変容支援を提供します。
無料で使えますか？
はい、研究期間中は完全に無料です。広告も一切表示されません。
対象年齢は？
主にAYA世代（15〜39歳）の小児がん経験者を対象にしています。18歳未満の方は保護者の同意のうえご利用ください。
どのデバイスで使えますか？
スマートフォン（iOS/Android）、タブレット、PC、いずれもブラウザで動作します。Chrome / Safari / Edge の最新版を推奨します。
⚔ 記録のしかた
運動の強度は何を選べばいい？
体感で選んでOKです。
  • 低：散歩、ストレッチ、ヨガなど（息は乱れない）
  • 中：早歩き、軽いジョギング、筋トレ（少し息が弾む）
  • 高：ランニング、HIIT、スポーツ（しっかり息が上がる）
強度が高いほどXPが多くもらえます。
食事はどこまで細かく記録すればいい？
カテゴリ（野菜・果物・タンパク質・主食）と、可能ならサブタイプ（魚・大豆・鶏など）を選べばOKです。完璧を求めず、「だいたい」で続けましょう。
記録を忘れた日があったらどうなる？
大丈夫です。連続記録（ストリーク）はリセットされますが、いつでも再開できます。「火を絶やさないこと」より「燃やし直すこと」のほうが大切です。
過去の日付の記録はできますか？
現在は「今日の記録」のみ対応しています。連続性が研究データとして重要なため、当日入力をお願いしています。
QOLの「体調・気分・活力」って何が違う？
  • 体調：身体の調子（痛み・だるさ・睡眠）
  • 気分：感情面（楽しい・落ち込み）
  • 活力：エネルギー量（やる気・元気さ）
3つを分けることで、傾向が見えやすくなります。
🌙 AIコーチ「ルナ」
ルナってどんなコーチですか？
GROWモデル（Goal/Reality/Options/Will）に沿って、目標を一緒に整理してくれるAIコーチです。指示するのではなく、あなたが自分で気づき、決められるよう問いかけます。
ルナが治療や薬の助言をしてくれる？
いいえ。ルナは医療助言を提供しません。診断・治療・服薬に関わる質問は、必ず主治医・専門家にご相談ください。
ルナは生活習慣・モチベーション・目標設計のサポートを行います。
会話内容は誰が見られる？
あなた自身、および研究の倫理審査を経た研究者のみが参照可能です（匿名化された形で解析）。第三者・他のユーザーには一切見えません。Gemini API への送信は、応答生成のみに使用され、AIの学習には使われません。
ルナの返答が遅い／返ってこない
ネットワーク状況や、Gemini API側の混雑が原因の可能性があります。
  1. Wi-Fi/通信状況を確認
  2. 1分ほど待って再送信
  3. ブラウザを再読み込み
それでも改善しない場合は、不具合タブの問い合わせフォームから連絡してください。
🎮 ゲーム要素
XPやレベルは何のため？
行動変容理論にもとづく「短期的フィードバック」です。記録のたびに小さな達成感を得られることで、習慣の継続率が上がるとされています。
冒険（アドベンチャー）はどう進めばいい？
12週間で1周する設計です。第1〜4章はチュートリアルと成長ボス、第5〜11章は7つの試練（七つの大罪）、第12章は最終ボスです。記録を続けることで自動的にダメージが蓄積し、ボスを倒せます。
実績（アチーブメント）は何種類？
運動・栄養・連続記録・QOL記録などのカテゴリで20種類以上を用意しています。解放されると画面上部にトーストが表示され、コンフェッティが舞います🎉
ログインボーナスは何日で何がもらえる？
  • 3日連続：+20XP
  • 7日連続：+50XP
  • 14日連続：+100XP
  • 28日連続：+300XP
招待コードって？
設定画面で6文字の招待コードを発行できます。友人がこのコードを「コードを使う」欄に入力すると、互いに+100XPがもらえます。研究としては、紹介経路の追跡にも使われます。
💾 データと同期
機種変更したらデータは引き継げる？
はい。Firebase認証（メール+パスワード）でログインすれば、新しい端末でも全データが復元されます。ローカルストレージのみのデータ（端末別の表示設定など）は引き継がれません。
オフラインでも使える？
記録の入力はオフラインでも可能で、一時的にローカルに保存されます。次にオンラインになった時点で自動的にクラウド同期されます。
データをエクスポートできますか？
はい。設定画面から CSV または JSON 形式でダウンロード可能です。研究を離脱する際にも、ご自身の記録を保存できます。
アカウントを削除したい
設定画面の「ローカルデータをクリア」でこの端末からは消去できます。サーバ上のデータも完全に削除したい場合は、研究問い合わせ窓口までご連絡ください。倫理規程に基づき、速やかに削除します。
🛠 不具合・トラブル
画面が真っ白／読み込まれない
  1. ブラウザの再読み込み（Ctrl/Cmd + R）
  2. キャッシュ削除（設定 → プライバシー → 閲覧データ削除）
  3. シークレットモードで開いて再現確認
  4. 別のブラウザ（Chrome / Safari / Edge）で試す
スマホで右側が切れて見える
ブラウザの拡大率設定が原因の可能性があります。設定→「サイトのズーム」を100%にして、再読み込みしてください。それでも直らない場合は、設定→アクセシビリティ→「文字を大きく」をオフにしてみてください。
バイブレーション・効果音が鳴らない
設定画面で「効果音」「バイブレーション」がオンになっているか確認してください。iPhoneの場合、サイレントモードではバイブが鳴らない仕様です。
通知が来ない
設定画面で通知をオプトインし、ブラウザ側でも通知許可を出してください。iOSの場合、PWAをホーム画面に追加した状態でのみ通知が届きます（iOS 16.4以降）。
記録が消えた／反映されない
  1. ログインしているか確認（右上のアカウントアイコン）
  2. ブラウザを再読み込み
  3. 30秒ほど待ってから再確認（同期に時間がかかる場合があります）
それでも消えている場合は、不具合タブの連絡先までご報告ください。
🔬 研究参加について
この研究の目的は？
小児がん経験者の健康行動継続を、ゲーミフィケーションとAIコーチで支援できるかを検証する研究です。修士論文として公表予定で、結果は学会・論文の形で社会に還元されます。
参加するとどんなメリット／デメリットがある？
メリット：自分の健康習慣を可視化できる、AIコーチに相談できる、研究成果として将来の患者さんに役立つ。
デメリット：記録に時間がかかる、画面を見る時間が増える、データを研究目的で利用される。
いつでも研究離脱が可能で、不利益はありません。
研究を途中でやめたい
いつでも自由にやめられます。設定画面または問い合わせフォームから「研究離脱」をご連絡ください。離脱後、ご希望に応じてデータを削除します。
プライバシーは守られますか？
はい。詳細はプライバシーポリシーをご覧ください。データは個人を特定できない形で集計・解析されます。
該当する質問が見つかりませんでした。
