## 3. デプロイ手順（Cloudflare 未接続の場合のみ）

`CLOUDFLARE_MIGRATION.md` に詳細手順があります。概要：

1. [Cloudflare ダッシュボード](https://dash.cloudflare.com) → Workers & Pages → Create → Pages
2. GitHub リポジトリ `rainybrainch/pediatric-cancer-support` を選択
3. Project name: `aya-health-quest`、Build command: 空欄、Output: `/`
4. Save and Deploy（1〜2分で完成）
5. Settings → Environment variables → `GEMINI_API_KEY` を追加（AIルナの動作に必要）
6. Retry deployment
