改修サマリ

追加ファイル（プロジェクトルート直下）

  • vitalia-rpg.css — RPGオーバーレイCSS（5テーマ：水彩／黄昏／星夜／深海／羊皮紙）
  • vitalia-rpg-fx.js — XPバースト・パーティクル・クエスト達成・レベルアップ・モンスター撃破
  • vitalia-rpg-battle.js — ターン制バトルモーダル（A1-A12モンスター × 4キャラ／攻撃・スキル・防御・撤退）
  • vitalia-rpg-tweaks.js — テーマ・演出密度の切替パネル（右側 ⚙ FAB）
  • vitalia-rpg-bind.js — 各ページ自動装飾＋既存UIへのフック

4 HTML ファイルへの注入（</head> 直前）

<link rel="stylesheet" href="./vitalia-rpg.css">
<script defer src="./vitalia-rpg-fx.js"></script>
<script defer src="./vitalia-rpg-battle.js"></script>
<script defer src="./vitalia-rpg-tweaks.js"></script>
<script defer src="./vitalia-rpg-bind.js"></script>

外部から呼べる API

  • VitaliaFX.xpFloat(x, y, amount) — 浮遊する「+XP」
  • VitaliaFX.sparkBurst(x, y, count, {color}) — パーティクル放射（gold/cyan/red/green）
  • VitaliaFX.questComplete({title, body, stats}) — 全画面クエストクリア演出
  • VitaliaFX.levelUp({level, portrait, subtitle}) — レベルアップシネマ
  • VitaliaFX.monsterDefeat(src, x, y) — モンスター撃破（撃破リストに自動保存）
  • VitaliaBattle.start({foeId, playerId}) — 戦闘画面起動
  • VitaliaFX.set({theme, density}) — 設定変更（自動で保存）

既存UIへの自動フック

  • .qbtn / .sh-rec-btn クリック時 — 自動でXP浮遊＋スパーク
  • #goal-banner 表示時 — クエストクリアシネマ起動
  • #save-toast 表示時 — 軽量スパーク＋＋XP
  • .chapter-card — 四隅にフィリグリー装飾追加
